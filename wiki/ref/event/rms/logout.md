## Syntax

```javascript
// Front-end (remote origin)
const resp = await api.dispatchEvent('rms:logout', {});
```

## Description

Logs out the current user, destroying the session.

- **Event:** `rms:logout`
- **Class:** `Zolinga\Rms\Api\UserApi`
- **Method:** `onLogout`
- **Origin:** `remote`
- **Event Type:** `\Zolinga\System\Events\RequestResponseEvent`

## Parameters

None.
